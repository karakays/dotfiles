# Dotfiles

This is my cross-platform configuration. It is actively running on *Debian*, sometimes on *macOS* and occasionally on *Windows Subsytem for Linux*.

## System

| component | package
|--- | ---
| shell | bash 
| editor | vim-gtk3
| terminal | alacritty
| terminal-multiplexer | tmux
| wm | i3
| status-bar | i3blocks
| application-launcher | rofi

## Themes

| package | theme
|--- | ---

| tmux | custom
